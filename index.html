What is HTTPS?

Have you ever wondered what HTTPS stands for? Let me explain!

HTTPS stands for HyperText Transfer Protocol Secure. It's used for sending data between the web browser and the website in a safe way, unlike HTTP which also is a hypertext transfer protocol, but it's missing the safety aspect. A safe, or a encrypted transer of information, means that important data, passwords for example, is protected and private.
